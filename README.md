# Proyecto Parcial - Clase Sensor
Es un sistema básico que guarda sensores usando clases, un vector de la STL y todo organizado con CMake.
## Cómo compilarlo
1. Crea una carpeta build: `mkdir build && cd build`
2. Configura CMake: `cmake ..`
3. Compila: `make`
4. Ejecuta: `./`
